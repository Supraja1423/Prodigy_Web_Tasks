# Prodigy InfoTech Internship – Software Development

This repository contains projects completed during my internship at Prodigy InfoTech, focusing on Software Development using Python.

The tasks demonstrate practical implementation of problem-solving, user interaction, data handling, and real-world application development.

---

## Task 1 – Temperature Conversion Program
A Python program that converts temperature values between Celsius, Fahrenheit, and Kelvin.

---

## Task 2 – Number Guessing Game
An interactive game where the user guesses a randomly generated number with feedback and attempt tracking.

---

## Task 3 – Contact Management System
A program to manage contacts with features to add, view, edit, and delete contact details.

---

## Task 4 – Sudoku Solver
A Sudoku application built using Streamlit with:
- Grid-based interface
- Solve functionality
- Input validation
- Clear and Reset options

---

## Task 5 – Web Scraping
A Python script that extracts product details (name, price, rating) from a website and stores the data in a CSV file.
